# Pilgrim on the Path

Real voices from the path. Each episode is one walk, recorded by a pilgrim and narrated by a voice guide.

**[podcast.pilgrimapp.org](https://podcast.pilgrimapp.org)**

A walking podcast presented as a vertical journey. Each episode is a goshuin-style seal on a winding path. Scroll to explore, click to listen.

Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step. Episodes are defined in `episodes/episodes.json` and rendered as an interactive journey map with seasonal themes, scroll-reactive ambient sound, and a real-time moon phase.

Part of [Pilgrim](https://pilgrimapp.org) — mindful walking.
